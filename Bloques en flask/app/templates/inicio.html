class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad   = edad

    def saludar(self):
        print(f"Hola, soy {self.nombre}")

# Instanciar un objeto
p = Persona("Daniel", 20)
p.saludar()
▶ Hola, soy Daniel

Objetivo

Mapear el compendio completo de Programación Orientada a Objetos en Python de forma interactiva en el navegador, eliminando la barrera de la terminal y permitiendo experimentar con cada concepto en tiempo real.

Metodología

Cada bloque contiene tres laboratorios independientes. Los parámetros se capturan desde formularios web, se procesan en el servidor mediante clases Python reales y la salida se muestra en una consola simulada.

Enfoque POO estricto

Todo el backend sigue obligatoriamente el paradigma orientado a objetos. Cada acción instancia una clase, procesa su estado interno y retorna resultados a través de métodos bien definidos.
